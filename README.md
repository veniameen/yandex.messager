# Yandex Praktikum Messager
Messenger web app using 
* Handlebars
* Express
* Typescript
* Sass

## Building and running
⚡ Installation
```
npm install
```
🐣 Running
```
npm start
```
Your app should now be running on http://localhost:3000/

🌳 Building
```
npm run build
```

📕 Eslint
```
npm run eslint
```

📗 Stylelint
```
npm run stylelint
```
